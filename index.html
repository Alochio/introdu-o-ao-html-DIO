O que são Tags:

Voltar

Tags são elementos utilizados em linguagens de marcação (como HTML, XML, Markdown, etc.) para definir a estrutura e o significado do conteúdo em um documento. Elas são delimitadas por caracteres especiais, como "<" e ">", e são usadas para marcar diferentes partes do texto, indicando como essas partes devem ser interpretadas ou apresentadas.

Por exemplo, em HTML, as tags são usadas para indicar diferentes elementos em uma página da web, como títulos, parágrafos, imagens, links, etc. Cada tag tem um propósito específico e define como o conteúdo dentro dela deve ser tratado pelo navegador ou outro software que interpreta o código.

h1 ao h6:

As tags do h1 ao h6 são tags que alteram o tamanho e a importância do texto que será escrito. Quanto mais perto do h1 maior e mais importante é o texto escrito dentro dessa tag, por isso usamos o h1 normalmente somente para o título da página.

p:

A tag p é utilizada para inserir um parágrafo dentro do seu site. Igual neste exato momento, aqui está sendo inserido um texto que está dentro da tag p.

Para usar a tag p basta somente iniciar e fechar a tag "<""p"">" "<""/p"">" escrevendo entre o inicio e o fim da tag o texto que deseja wue fique dentro do parágrafo.

Essa tag deve ser chamada dentro da tag p, para que a marcação deva se realizar dentro do parágrafo por exemplo.
